Running C++ in a browser via codepad

Running Javascript in a browser is easy, since that is what it is designed for. Python can be run in a browser via Jupyter. You might be able to run Java in a browser via an applet, but that solution is practically deprecated. C++ won't run in your browser. However, you can demonstrate your progams running in a browser using codepad.org. As far as I can tell, it runs your code on the server, and sends the results back. This is useful if you want to allow visitors to your website to view and even play around with your code. You will need to create a free account.
