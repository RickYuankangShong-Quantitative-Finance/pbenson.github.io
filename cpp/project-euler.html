Project Euler with C++

If you want to learn a language effectively, and be able to demonstrate that you've learned the language, you must practice. Project Euler provides an abundance of practice problems that will help you get used to the syntax of the language, and once you have solved a problem, you are able to see how others have solved it. Along the way, you should try to take advantage of C++'s collection classes, and write your own classes. The folks who've put Project Euler together ask that you not publish your solutions. However, you can invent and publish your own problems inspired by Project Euler.
