Configuring your IDE to find your data files

Xcode

Check out this stackoverflow discussion. For example:

Put your .txt files in the same directory where your main.cpp file is (or anywhere you like).
In Xcode go to Product > Scheme > Edit Scheme > Run test (on the right) > Options (middle top).
Down under Options, check “Use custom working directory” and set it to the directory where your .txt files are located.
