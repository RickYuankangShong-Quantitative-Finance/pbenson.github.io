C++ Basics

Comparing C++ to Python et al

You've all learned at least some Python, and it is beneficial to understand some of the key differences.

  1. C++ is statically typed. This means you must declare the type of all variables. Python infers the type of a variable at runtime when the variable is assigned a value.
  2. C++ programs are typically a collection of C++ source files (which might have .cpp or .cc extensions), some header files (e.g. .h files), and possible others. Python files are .py, or in the case of Jupyter notebooks, .ipynb files.

Physical program structure in C++

C++ programs consist typically of a file named main.cpp, and a collection of other files with a .cpp extension, with each file optionally having a corresponding .h file. Our programs will be initially small, and will consist of a single file, main.cpp.
