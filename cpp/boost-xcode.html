boost libraries in Xcode on Mac OS X

Installing boost and configuring it in your Xcode project

Boost is regularly updated, and you can see the current release number on the boost home page. As of this writing, it is on version 1.62.0. The best installation instructions I've found for the Mac are here on stackexchange (use the most highly rated solution).

In a nutshell:

  1. Launch Terminal, e.g., by clicking on Spotlight and typing "Terminal."
  2. Install homebrew.
  3. With homebrew installed, you can install the current release of boost by running this on the command line: 
    brew install boost
  4. In XCode, select the project file in the left sidebar (top most), and select it again in the project pane.
  5. Select the Build Settings tab, and select All.
  6. Under Search Paths, add the header search path /usr/local/Cellar/boost/1.62.0/include/ and check the box. The path should now show in the Build Settings.
