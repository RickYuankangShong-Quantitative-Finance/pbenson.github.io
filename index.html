QFRM boot camp

In the QFRM boot camp, you will develop your personal web presence with your umich page, your GitHub user page linked to GitHub projects (a page like this, but yours, not mine), and Jupyter notebooks that demonstrate your Python skills as they relate to finance.

Bootcamp requires that you have Atom, Anaconda, and GitHub Desktop installed.
