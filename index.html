QFRM student websites

Students in the QFRM program at the U of M create websites to describe their graduate work. A quick way to create a site could be to use something like Google Sites or Wordpress.org, but these solutions are blocked in China. You can test a website using a site like blockedinchina.net.

GitHub Pages sites are free, and are not blocked as far as I know. Students can roll their own websites, or search online for a template, or they can start with a template from below. These were orginally inspired by CSS Zen Garden. All of the templates have exactly the same HTML, which is customized by the student using a text editor like Atom. As long as the student does not change class names in the HTML document, they can easily change the look of their site by swapping a new styles folder.
