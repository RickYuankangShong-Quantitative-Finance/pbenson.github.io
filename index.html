QFRM Bootcamp agenda

Below is the agenda we follow to write and refine resumes, build a University web page and personal website, and learn some financial concepts while developing programming skills. Not necessarily in that order.

  1. Overview of your online employment portfolio
    1. Your U of M home page.
    2. resume
    3. LinkedIn page
    4. personal website hosted with GitHub Pages. You'll need to create a free GitHub account. Use an appropriate name for your account, as it will be visible to potential employers.
  2. Install tools: GitHub client, Anaconda, Atom. We assume you have a word processor such as Google doc or MS Word.
    1. Create a GitHub account with a professional username.
    2. Download and install GitHub Desktop.
    3. Install Anaconda if you don't already have it. We prefer the 2.7 edition of Python, and recommend the graphical installer.
    4. Follow step one of the Conda 30 minute test drive
    5. Follow step four of the Conda 30 minute test drive.
    6. Launch Jupyter Notebook. You may want to create a special directory on your laptop for your Jupyter notebook files which you can maintain with GitHub.
  3. Python Basics
    1. From the GitHub client, create a GitHub repository , and name it python-basics-project.
    2. Create a README.md file using Atom in your python-basics-project directory. Add this text to it: “These are examples of programs I’ve written while learning Python. The implementation is in Jupyter notebooks."
    3. Create a Jupyter notebook, and name it python-basics-notebook. We'll use it for the following...
    4. simple data types: int, float, boolean, string
    5. control structures: for loops, if/else
    6. collection data types: lists, dictionaries
    7. for loops with collections
    8. functions
    9. commit your work via the GitHub client, and sync.
  4. Additional Python practice
    1. From the GitHub client, create a GitHub repository, and name it python-secret-project.
    2. Create a README.md file using Atom in your python-secret-project directory. Add this text to it: “These are programs I’ve written to practice Python. I am striving for concise, readable, and sufficiently effciient code."
    3. Create a Jupyter notebook, and name it python-secret-notebook. We'll use it for problems in Project Euler.
    4. commit your work via the GitHub client, and sync.
    5. Other practice problems you might consider...
      1. Google Code Jam
      2. Programming Praxis
  5. Python in Finance
    1. From the GitHub client, create a GitHub repository , and name it python-finance-project.
    2. Create a README.md file using Atom in your python-finance-project directory. Add this text to it: “These are programs I’ve written to solve quantitative finance problems. I use the numpy and pandas libraries."
    3. Create a Jupyter notebook in your project, and name it zero-coupon...commit.
    4. Create a notebook named black-merton-scholes... and commit.
    5. Create a notebook named price-model... and commit.
    6. Create a notebook named historical-simulation... and commit.
    7. Create a notebook named brownian-motion... and commit.
    8. Create a notebook named monte-carlo-simulation... and commit.
    9. Create a notebook named risk-measures... and commit.

QFRM student websites

We will create a personal website to describe and provide evidence of the depth and quality of your work. You can create a hosted site easily using Google Sites or Wordpress.org, but these solutions are blocked in China, which is undesirable if you intend to apply for jobs there. You can test the accessibilitiy of a website via blockedinchina.net.

GitHub Pages sites are free, and are not blocked. You can roll your own websites, or search online for a template, or you can start with a template from below. These were orginally inspired by CSS Zen Garden. All of the templates have exactly the same HTML, which is customized by the student using a text editor like Atom. As long as the student does not change class names in the HTML document, they can easily change the look of their site by swapping a new styles folder.

  2. Zen 214
  3. Zen 220

QFRM Python

Through the development of powerful libraries such as numpy, pandas, scipy and others, Python has become a popular tool on Wall Street. In the QFRM bootcamp, we practice using these Python to solve math problems, and use the libraries to solve financial problems.
