How to contribute to umich-quant

back to quant-projects

First, you'll need to get the project up and running on your machine. And for that, you need to be properly installed, so check out Getting Started. Now, since you've followed the instructions carefully, you've installed an IDE and the boost library, and configured your IDE to find the boost libraries. So you can create a command line project, replace the source of the main.cpp file with the code below, and compile and run it successfully, with a return code of 0.

Hosting on GitHub

A version of the project can be cloned from GitHub. For more information on using GitHub in a team setting, read this.

Here are the steps:

  1. If you haven't already, create a GitHub account, and verify the account by responding to the verification email you were sent when you created the account. Install GitHub Desktop.
  2. Clone the umich-quant project, using "Open in Desktop" option. Accept the default directory, which is Documents on Mac OS X.
  3. Navigate to your project, and if you are using Xcode, find the MultiFactorMonteCarlo.xcodeproj file, and launch it.
  4. Configure your project to find your data file.
  5. In your main.cpp, put a breakpoint on the last line of code (return 0;), and hit the run button. Your program should stop at the breakpoint.

The GitHub version includes a unit testing target, based on Catch.
