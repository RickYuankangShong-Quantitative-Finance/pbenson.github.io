Getting Started

There are many C++ IDEs to choose from. For most, the IDE of choice on Windows is Visual Studio, and the preferred Mac IDE is Xcode. Both are free.

  1. Download and install the community edition of Visual Studio (Windows) or install Xcode from the Mac App Store.
  2. Create an account on codepad.org.
  3. Create your first project. In Xcode, select File->New->Project, and select Command Line Tool.

For additional help with C++ in Xcode, see Jesse Daugherty and Chris Wendel's site, and the Xcode Notes for EECS 381.
