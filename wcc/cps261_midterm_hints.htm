Midterm Hints and Procedures

Procedures

  1. This test consists of short answer questions (i.e. multiple choice). There are no programs to write and you are not allowed to use Eclipse on this test, because Eclipse could answer many of the questions.
  2. The test is open book, open notes, and open web.
  3. If you need lots of time for tests (more than 4 hours), please take it in the testing center.
  4. If you don't take the test in the testing center, then there will be a time limit on the test of 4 hours which should be plenty of time for most students. On the test, I want to know if you understand the Java in the test and not whether you can type the Java into Eclipse, run it and discover the answer(most of you would never do this, but I have had students who do this kind of a thing). The test will not turn off at 4 hours, but the time you took for the test will be known.

  • MouseListener and MouseMotionListener interfaces and associated routines
  • MouseAdapter class
  • Exceptions - Make sure you know about checked and unchecked exceptions ... i.e. RutimeExceptions, Exceptions, common exceptions like NullPointerException, ArrayIndexOutOfBounds, etc. In the notes and videos I worked through these situations.
  • Catching Exceptions, throwing exceptions, processing exceptions
  • Scanner class used with file input
  • ObjectOutputStream and ObjectInputStream and the Serializable interface
  • File class
  • From the Animation notes we covered simple Thread concepts: Runnable Interface, start, sleep methods
